Roadmap

Planned direction for the AshForge ecosystem.

Short-term

  • Polish current add-ons and settings pages.
  • Add documentation for each release.
  • Continue improving AshForge Hub as the central launcher.

Possible future add-ons

  • RSS Forge
  • Weather Forge
  • Archaeology Forge
  • Unfavorites
  • File Cabinet
  • Session Savior

Long-term

  • Shared AshForge integration between supported add-ons.
  • Consistent help, privacy, and support resources.
  • Keep each add-on focused and optional.

AshForge Addons enhances Firefox with focused tools that build on existing browser features instead of replacing them.

Reviving the best. Reforging the rest.

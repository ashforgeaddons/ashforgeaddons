Support

Need help with an AshForge add-on? Use the steps below.

Email Support

Use your AshForge support email for bug reports, feature requests, and questions.

ashforgeaddons@gmail.com

Before reporting a bug

  • Include the add-on name and version.
  • Describe what you clicked or changed.
  • Include any Firefox or AMO error message.
  • Mention whether restarting Firefox helped.

Feature requests

New ideas are welcome, especially when they make an existing browser feature easier to use.

AshForge Addons enhances Firefox with focused tools that build on existing browser features instead of replacing them.
